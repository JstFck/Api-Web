To use API write GET-request on this URL: http:/127.0.0.1:5000/api

  • apikey: apikey, which required to use API
  • city: city name, which required to get info about city
  • um units of measurement: um=m metric (SI) / um=u USCS (used by default in US)
  • lang supported languages: am ar af be bn ca da de el et fr fa gl hi hu ia id it lt mg nb nl oc pl pt-br ro ru ta tr th uk vi zh-cn zh-tw
  • days return specified day (max 3)
  • wind return wind speed in m/s if wind=M

Example of request and response

  • Request: http://127.0.0.1:5000/api?apikey=12345&city=moskow
  • Response: { "moskow": [ "Overcast", "+10(8) °C", "↘ 22 km/h" ] }
